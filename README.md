# YC2Classics

A server backend for some article storage

---

# Build and run
## Docker Compose

1. `git clone https://github.com/ZZSZ-YCT/YC2Classics.git`
2. `cp example.env .env`
3. Edit `.env` file as you want
4. `docker compose up`

## Gradle

1. Edit your environment variable according to `example.env`
2. `gradle runFatJar`
